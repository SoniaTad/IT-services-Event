Docker

What is docker:

Docker is an open-source platform used to develop and run encapsulated application within containers.

Why use docker:

Docker containers are appealing for many reasons, but they mainly are:

  • Portable
  • Flexible
  • Sharable
  • OS-independent
  • Fast

How to use Docker ?

Using docker is quick and simple . After creating a dockerfile and building the image , an instance of that image (container) can then be run.

Kubernetes

What is kubernetes:

Kubernetes is an open-source container orchestrater . In other words, it deploys , runs, and manages containers in the cloud and on-premises. Assuring that the number of resources needed is always satisfied no matter the demand .

Why use it:

Kubernetes has some serious advantages which play a massive role in its constantly growing popularity. Naming a few:

  • Automation and load balancing
  • Scalability
  • High availability
  • Security
